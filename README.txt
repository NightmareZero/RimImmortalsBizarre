[h1] 边缘仙路 道诡异仙 [/h1]

[h2] 简介 [/h2]

边缘仙路-道诡异仙扩展，是一个基于《道诡异仙》的mod，增加了一些新的内容，包括几条法门树，和一些新的法术。

目前该mod还在开发中，欢迎大家提出建议。

未来，这个mod将包含 4 条法门树，分别是 “袄景教”， “坐忘道”， “正德寺”，“白莲教”

[h2] 派系介绍 [/h2]

[h3] 袄景教 [/h3]

袄景教是一个擅长操纵血肉的派系，他们的法术大多是以牺牲自己的血肉为代价，来获得强大的力量。
同时，他们也擅长治愈法术，用于修复自己的伤口。

[b] 典型技能 [/b]

[u] 手指 [/u]：献祭一根手指，造成扇形酸液伤害，并且血肉会如附骨之蛆一样，给敌人造成剧烈的痛楚。

[u] 手 [/u]：献祭一只手，对小范围造成高额伤害

[u] 火疗术 [/u]：使用火焰灼烧肉体，为伤口止血，还能同时治愈一部分。

[u] 再生 [/u]：献祭自己一部分内脏，修复目标缺失的部位。

[u] 苍羌登阶 [/u]：润置五行, 换取巴虺的注视，内心和身体越痛苦，成功率越高，会对内脏造成大量伤害

[h3] 坐忘道 [/h3]


坐忘道是一个专门擅长欺骗的派别，他们的大部分法术都是专注于欺骗对手的感官，并利用这一点来获得一种名为“非罡”的资源，可以用来释放更强大的法术。
随着更多的欺骗行动，偶尔会吸引来 “心素”。他们的眼睛，在坐忘道身上，可以逐渐生成先天一气
坐忘道的社交越高，对方的智力越低，对方对你的好感度越高，欺骗概率就越高，尝试先打好关系吧

[b] 典型技能 [/b]

[u] 花言巧语 [/u]： 巧妙的恭维，让人心生好感

[u] 混珠诳取 [/u]： 消耗少量，幻化成目标梦寐以求的宝物，目标会使用自己的物品来交换

[u] 李代桃僵 [/u]： 消耗少量非罡，将一个人身上的负面效果转移到另一个人身上

[u] 镜中之象 [/u]： 消耗大量先天一气，从虚幻与现实的镜子中取出目标的影子，由于镜子只映照出一半，所以目标会有残缺，会产出大量非罡。

[u] 罔天宝诰 [/u]： 消耗巨量非罡，将司命“斗姥”暂时拉入凡间，你的敌人会因为目睹到司命，陷入到心想事成的幻觉之中，而身体由于失去控制倒地。


[h3] 正德寺 [/h3]

到了正德寺这个法门，在书中体现的篇幅就比较少了，主要是修行血肉和对于百姓的慈悲，(当然还有比较Rjw的部分，但是我不打算碰这相关的东西)

于是做正德寺的设计时，我想到了，当‘大傩’世界，在全书结束时，恢复成一个正常世界的时候，是否会有其正常的修行方式。
但是相关的想法太多，我干脆将正德寺设定为，初始的时候只有少量能力，根据技能的释放和自身的修行，逐渐增强，获得新的能力

[b] 初始技能 [/b]

[u] 参禅 [/u]： 在行走坐卧中，参悟禅理，降低移动速度，但是修行进度会随时间上升, 其他特殊能力会随着修行逐渐解锁

[u] 禅音 [/u]： 发出一声蕴含禅意的声音，令人幡然醒悟。解除迷茫和狂暴状态，同时略微增加意识和操作。这会让你更接近 “明心” 路线

[u] 布施 [/u]： 割肉饲鹰 补充目标的饱腹和休息，同时增强目标的愈合速度 这会让你更接近 “舍身” 路线

[u] 突破 [/u]： 选择一个可用的路线进行突破(每次突破成功，都会获得一个新的仿生体，每条路线最多有8个仿生体，互相之间部位冲突)

[b] 修行路线 [/b]

[u] 克己 [/u]： 克己之法，不求法门神通，专注修持自身，随着修行的进行，你的工作效率研究速度等都会提升

[u] 明心 [/u]： 为向众生阐明佛理，普渡众生，缺乏杀伤能力，以控制和保护为主

[u] 舍身 [/u]： 以身殉道，为众生解难，但是，也要小心，这里不是'大傩'，回应你的也不一定是'司命'

[h3] 兵家 [/h3]

这是最近临时冒出来的想法，应该是一个小品级的，技能偏少的法门
是较为廉价的前排，会有依靠染煞兵器传承修为的能力
